# Analisis Pola Transaksi Shopee Food

Aplikasi Streamlit untuk penelitian skripsi:

**ANALISIS POLA TRANSAKSI SHOPEE FOOD MENGGUNAKAN METODE K-MEANS CLUSTERING BERDASARKAN DATA PEMESANAN PADA TOKO BUFFET THE PADANG PASIR**

## Fitur

- Dashboard
- Kelola Data (CRUD)
- Import CSV
- Preprocessing
- K-Means Clustering
- Hasil Clustering
- Download CSV
- Halaman Tentang

## Instalasi

```bash
pip install -r requirements.txt
streamlit run app.py
```
